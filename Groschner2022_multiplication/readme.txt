This collection contains data and code used in the manuscript "A biophysical account of multiplication by a single neuron" by Lukas Groschner, Jonatan Malis, Birte Zuidinga, and Alexander Borst.

Analyses and simulations were performed using the Anaconda distribution of Python (https://docs.anaconda.com/anaconda/install/index.html).

Download the collection and navigate to the respective path in the terminal (Mac) or in an Anaconda Prompt (Windows).

Run the following code to create the environment:
conda env create --file environment.yml

Activate the environment:
conda activate MultiplicativeDisinhibition

Start JupyterLab:
jupyter lab

In JupyterLab, navigate to the desired .ipynb file and run the code.

For questions, please contact Lukas Groschner (groschner@neuro.mpg.de).
